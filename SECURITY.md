# Security Policy

## Supported Versions of shellnotes



| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   |        yes         |
| 1.0.x   |        yes         |
| 0.1.x   |        no          |


## About updating

Every new update is uploaded in this page.
If you want to check your version, type "shellnotes -v" and check if the version is supported.
